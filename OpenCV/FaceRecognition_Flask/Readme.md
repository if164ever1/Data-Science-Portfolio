# OpenCV Face Recognition with Flask

This project aims to implement face recognition using OpenCV and Flask.

## Setup

1.  **Create a Virtual Environment**

    It's recommended to use a virtual environment to manage project dependencies. From the project root directory, run:

    ```bash
    python -m venv venv
    ```

2.  **Activate the Virtual Environment**

    -   **Windows (PowerShell)**:

        ```bash
        .\venv\Scripts\activate
        ```

    -   **Windows (Command Prompt)**:

        ```bash
        .\venv\Scripts\activate.bat
        ```

    -   **macOS/Linux**:

        ```bash
        source venv/bin/activate
        ```

3.  **Install Dependencies**

    Once the virtual environment is activated, install the required packages using pip:

    ```bash
    pip install -r requirements.txt
    ```
