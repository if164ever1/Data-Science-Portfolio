# OpenCV Practical Python and OpenCV Learning Project

This repository contains a collection of Jupyter notebooks and associated files for learning practical image processing with Python and OpenCV.

## Project Structure

- `getting_and_setting.ipynb`: Notebook exploring basic operations like getting and setting pixel values.
- `load_display_save.ipynb`: Notebook demonstrating how to load, display, and save images.
- `requirements.txt`: Lists the necessary Python packages to run the notebooks.
- `1.webp`, `img3.png`, `new_image3.jpg`: Sample image files used in the notebooks.

## Setup

To get started with this project, you'll need to set up your Python environment and install the required libraries. It is recommended to use a virtual environment.

1. Clone the repository:
   ```bash
   git clone <repository_url>
   cd "Practical Python and OpenCV"
   ```

2. Create and activate a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   .\venv\Scripts\activate
   ```

3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

After setting up the environment, you can launch Jupyter Lab or Jupyter Notebook to explore the `.ipynb` files.

```bash
jupyter lab
# or
jupyter notebook
```

## Key Concepts

This project covers fundamental concepts in image processing and computer vision using OpenCV, including:

- **Image Loading and Saving**: How to read various image formats and write processed images back to disk.
- **Pixel Manipulation**: Accessing and modifying individual pixel values or regions of interest.
- **Image Display**: Displaying images and video streams using OpenCV functions.
- **Basic Image Operations**: Understanding image properties like dimensions, channels, and data types.
- **Image Rotation**: Techniques for rotating images by various angles.
- **Image Translation**: Shifting images along the X and Y axes.
